## 使用方式(不推荐的)

> **Warning**
> 正常来说，你不需要全部采用我的配置，这很危险！！！它会覆盖你原有的配置，建议你阅读 `backup` 这个目录，摘选可能适合你的配置。

### 1. 克隆仓库

```bash
git clone git@github.com:Wxh16144/dotfiles.git && cd dotfiles
```

### 2. 安装依赖

```bash
npm install
```

### 3. 应用

```bash
npm run restore
```
